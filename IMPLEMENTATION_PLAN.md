## 3. 风险与应对
1. 风险：OpenRouter key 延迟。
   - 应对：默认 mock，真实调用接口预留。
2. 风险：Qwen 输出不稳定。
   - 应对：固定 JSON schema + 解析失败回退 top1。
3. 风险：图片生成耗时大于 120s。
   - 应对：限制单次只生成 1 张，保留重试上限 1。
4. 风险：相似度阈值误伤。
   - 应对：审计记录阈值命中样本，后续再调参。
