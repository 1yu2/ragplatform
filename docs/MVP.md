# MVP Scope

## Must-have
- PDF upload
- Parsing + chunking
- Embedding + vector index
- Retrieval + rerank
- Answer + citations
- Team-level RBAC (basic)

## Nice-to-have (post-MVP)
- Multi-source ingestion (web, DB)
- Agent multi-hop retrieval
- Rerank A/B
- Online eval dashboard

## Milestones
1. Upload -> index -> search
2. Chat + citations
3. Rerank + hybrid retrieval
4. Evaluation (RAGAS/TruLens)
5. Permissions + audit
