# 博客文章写作技能

此技能提供了一个结构化的工作流程，用于创建能够教育并吸引读者的优质博客文章。

**重要：所有博客内容必须用中文撰写，包括标题、正文、结论等。使用中文标点符号（如，。！）**

## 何时使用此技能

当被要求以下操作时使用此技能：
- 撰写博客文章或文章
- 创建教程或操作指南
- 开发教育类长篇内容
- 撰写思想领导力文章

## 首先研究（必需）

**在撰写任何博客文章之前，你必须委托研究：**

1. 使用 `task` 工具，设置 `subagent_type: "researcher"`
2. 在描述中，指定主题和保存位置：

```
task(
    subagent_type="researcher",
    description="Research [TOPIC]. Save findings to research/[slug].md"
)
```

示例：
```
task(
    subagent_type="researcher",
    description="Research the current state of AI agents in 2025. Save findings to research/ai-agents-2025.md"
)
```

3. 研究完成后，在写作之前阅读研究结果文件

## 输出结构（必需）

**每篇博客文章必须同时包含文章和封面图片：**

```
blogs/
└── <slug>/
    ├── post.md        # 博客文章内容
    └── hero.png       # 必需：生成的封面图片
```

示例：关于 "AI Agents in 2025" 的文章 → `blogs/ai-agents-2025/`

**你必须完成两个步骤，不要询问用户！：**
1. 将文章写入 `blogs/<slug>/post.md`
2. 使用 `generate_cover` 生成封面图片并保存到 `blogs/<slug>/hero.png`

**必须生成封面图片，不要询问用户！**

**没有封面图片的博客文章是不完整的。**

## 博客文章结构

每篇博客文章应遵循以下结构：

### 1. 开篇（开头）
- 以引人入胜的问题、统计数据或陈述开始
- 让读者想要继续阅读
- 保持 2-3 句话

### 2. 背景（问题）
- 解释为什么这个主题重要
- 描述问题或机会
- 与读者体验联系起来

### 3. 主要内容（解决方案）
- 分成 3-5 个主要部分，用 H2 标题
- 每个部分涵盖一个关键点
- 在有帮助的地方添加代码示例、图表或截图
- 使用列表

### 4. 实际应用
- 展示如何应用概念
- 如有需要，包含分步说明
- 提供代码片段或模板

### 5. 结论和行动号召
- 总结关键要点（最多 3 点）
- 以明确的行动号召结尾
- 链接到相关资源

## 封面图片生成

撰写文章后，使用 `generate_cover` 工具生成封面图片：

```
generate_cover(prompt="A detailed description of the image...", slug="your-blog-slug")
```

该工具将图片保存到 `blogs/<slug>/hero.png`。

### 编写有效的图片提示词

使用以下元素构建提示词：

1. **主体**：主要内容是什么？要具体和明确。
2. **风格**：艺术方向（极简、等距、平坦设计、3D 渲染、水彩等）
3. **构图**：元素如何排列（居中、三分法、对称）
4. **配色板**：特定颜色或情绪（暖色调、冷蓝和紫色、高对比度）
5. **光线/氛围**：柔和散射光、戏剧性阴影、黄金时刻、霓虹光晕
6. **技术细节**：宽高比考虑、用于文字叠加的负空间

### 示例提示词

**技术博客文章：**
```
等距 3D 插图，展示互联的发光立方体代表 AI 智能体，每个立方体有微妙的电路图案。立方体由发光数据流连接。深海军蓝背景 (#0a192f) 配 electric blue (#64ffda) 和柔和紫色 (#c792ea) 点缀。简洁极简风格，顶部留有大量负空间用于标题。专业科技美学。
```

**教程/操作指南：**
```
简洁平坦的插图，展示双手在键盘上打字，抽象代码符号向上飘动，变成灯泡和齿轮。柔和珊瑚色到浅桃色的暖渐变背景。友好、平易近人的风格。居中构图，留有文字叠加空间。
```

**思想领导力：**
```
人类侧影轮廓与几何神经网络模式融合的抽象可视化。分割构图 - 左侧有机水彩纹理过渡到右侧清晰的矢量线条。柔和鼠尾草绿和暖陶土色调。沉思、前瞻性的情绪。
```

## SEO 注意事项

- 在标题和第一段包含主要关键词
- 在全文中自然使用关键词 3-5 次
- 标题保持在 60 个字符以内
- 撰写元描述（150-160 个字符）

## 质量检查清单

完成前检查：
- [ ] 文章已保存到 `blogs/<slug>/post.md`
- [ ] 封面图片已生成到 `blogs/<slug>/hero.png`
- [ ] 开篇在前 2 句话内抓住注意力
- [ ] 每个部分都有明确目的
- [ ] 结论总结关键要点
- [ ] 行动号召告诉读者下一步做什么
