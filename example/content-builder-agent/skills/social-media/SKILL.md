# 社交媒体内容技能

此技能提供了创建引人入胜的社交媒体内容的指南，以推动参与和分享。

**重要：所有社交媒体内容必须用中文撰写。使用中文标点符号（如，。！）**

## 何时使用此技能

当被要求以下操作时使用此技能：
- 撰写 LinkedIn 帖子
- 创建 Twitter/X 线程
- 起草社交媒体公告
- 将博客内容重新用于社交平台

## 首先研究（必需）

**在撰写任何社交媒体内容之前，你必须委托研究：**

1. 使用 `task` 工具，设置 `subagent_type: "researcher"`
2. 在描述中，指定主题和保存位置：

```
task(
    subagent_type="researcher",
    description="Research [TOPIC]. Save findings to research/[slug].md"
)
```

示例：
```
task(
    subagent_type="researcher",
    description="Research renewable energy trends in 2025. Save findings to research/renewable-energy.md"
)
```

3. 研究完成后，在写作之前阅读研究结果文件

## 输出结构（必需）

**每个社交媒体帖子必须同时包含内容和图片：**

**LinkedIn 帖子：**
```
linkedin/
└── <slug>/
    ├── post.md        # 帖子内容
    └── image.png      # 必需：生成的视觉素材
```

**Twitter/X 线程：**
```
tweets/
└── <slug>/
    ├── thread.md      # 线程内容
    └── image.png      # 必需：生成的视觉素材
```

示例：关于 "prompt engineering" 的 LinkedIn 帖子 → `linkedin/prompt-engineering/`

**你必须完成两个步骤：**
1. 将内容写入相应路径
2. 使用 `generate_image` 生成图片并与帖子一起保存

**没有图片的社交媒体帖子是不完整的。**

## 平台指南

### LinkedIn

**格式：**
- 1300 字符限制（约 210 字符后显示"展开更多"）
- 第一行至关重要 - 要能抓住注意力
- 使用换行提高可读性
- 末尾 3-5 个话题标签

**语气：**
- 专业但个人化
- 分享见解和学习
- 提问以推动参与
- 使用"我"分享经历

**结构：**
```
[开篇 - 1 行引人注目的内容]

[空行]

[背景 - 为什么这很重要]

[空行]

[主要见解 - 2-3 个短段落]

[空行]

[行动号召或提问]

#标签1 #标签2 #标签3
```

### Twitter/X

**格式：**
- 每条推文 280 字符限制
- 较长内容使用线程（使用 1/🧵 格式）
- 每条推文最多 2 个话题标签

**线程结构：**
```
1/🧵 [开篇 - 主要见解]

2/ [支持观点 1]

3/ [支持观点 2]

4/ [例子或证据]

5/ [结论 + 行动号召]
```

## 图片生成

每个社交媒体帖子都需要引人注目的图片。使用 `generate_social_image` 工具：

```
generate_social_image(prompt="A detailed description...", platform="linkedin", slug="your-post-slug")
```

该工具将图片保存到 `<platform>/<slug>/image.png`。

### 社交图片最佳实践

社交图片需要在拥挤的信息流中以小尺寸工作：
- **醒目、简单的构图** - 一个清晰的焦点
- **高对比度** - 滚动时脱颖而出
- **图片中不要有文字** - 太小无法阅读，平台会添加自己的文字
- **方形或 4:5 比例** - 跨平台通用

### 编写有效的提示词

包含以下元素：

1. **单一焦点**：一个清晰的主体，不是复杂的场景
2. **醒目风格**：鲜明的颜色、强烈的形状、高对比度
3. **简单的背景**：纯色、渐变或微妙的纹理
4. **情绪/能量**：匹配帖子语气（鼓舞人心、紧迫、深思熟虑）

### 示例提示词

**见解/技巧帖子：**
```
单个发光灯泡漂浮在深紫色渐变背景上，灯泡由互联的金色几何线条组成，柔和的光线向外散发。简洁、醒目、高对比度。方形构图。
```

**公告/新闻：**
```
由彩色几何形状组成的抽象火箭飞船向上发射，带有粒子轨迹。干净的白色背景上使用明亮的珊瑚色和青绿色配色。充满活力、庆祝的氛围。醒目平坦的插画风格。
```

**发人深省的内容：**
```
两个重叠的半透明圆圈，一个蓝色一个橙色，在中心创造发光交集。代表协作或想法的交集。深炭灰色背景，柔和的空灵光晕。极简且沉思。
```

## 内容类型

### 公告帖子
- 以新闻开头
- 解释影响
- 包含链接或下一步

### 见解帖子
- 分享一个具体的学习
- 简要解释背景
- 使其可操作

### 提问帖子
- 问一个真正的问题
- 先提供你的看法
- 保持聚焦于一个主题

## 质量检查清单

完成前检查：
- [ ] 帖子已保存到 `linkedin/<slug>/post.md` 或 `tweets/<slug>/thread.md`
- [ ] 图片已与帖子一起生成
- [ ] 第一行抓住注意力
- [ ] 内容符合平台限制
- [ ] 语气匹配平台规范
- [ ] 有明确的行动号召或提问
- [ ] 话题标签相关（不是通用的）
